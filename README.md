⚙️ Instalasi & Setup
# Clone repo
git clone https://github.com/syaifulamri1625/sundaverse-v.3.0.git
cd sundaverse-v.3.0

# Install dependencies
npm install
# atau
yarn install

# Setup environment
cp .env.example .env

# Jalankan development server
npm run dev
# atau
yarn dev


Lalu buka aplikasi di browser:
👉 http://localhost:3000
